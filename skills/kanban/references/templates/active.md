# Active - 执行中任务

> 当前正在执行的任务（≤10条）

## 格式
```markdown
- [状态] T00X: [P优先级] [project:项目名] [category:类别] 任务描述
  - owner: 
  - depends-on: 
  - blocked-reason: (如果有)
  - updated: YYYY-MM-DD
```

## 状态说明
- `pending`: 待开始
- `in_progress`: 执行中
- `blocked`: 卡住了，需要人补充信息
- `done`: 完成
