# Active - 执行中任务

> 当前正在执行的任务（≤10条）

## 格式
```markdown
- [状态] T00X: [P优先级] [项目名] 任务描述
  - owner: 
  - depends-on: 
  - updated: YYYY-MM-DD
```

## 状态说明
- `pending`: 待开始
- `in_progress`: 执行中
- `blocked`: 卡住了
- `done`: 完成
