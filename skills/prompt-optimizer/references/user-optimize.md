# 用户提示词优化

用户提示词优化将模糊、简单的提示词转化为清晰、具体、有效的提示词。

## 模板类型

### 1. 基础优化 (user-prompt-basic)

**适用场景**: 简单快速的日常优化，消除模糊表达、补充关键信息

**触发方式**:
```
优化：帮我写个文章
基础优化：写个故事
```

**优化要点**:
- 模糊词汇识别与替换（如"好看"→具体描述）
- 信息补充（缺失的关键细节）
- 结构整理（逻辑清晰）
- 目标明确（模糊意图→具体目标）

---

### 2. 规划优化 (user-prompt-planning)

**适用场景**: 复杂任务场景，将模糊需求分解为具体执行步骤

**触发方式**:
```
规划：帮我做一个旅行计划
步骤化：完成这个任务
```

**优化要点**:
- 意图识别 → 核心目标
- 任务分解 → 子任务
- 步骤排序 → 执行顺序
- 细节补充 → 可执行性

**输出结构**:
```
# 任务：[核心任务标题]

## 1. 角色与目标
[专家角色] + [可衡量的最终目标]

## 2. 背景与上下文
[补充说明或"无"]

## 3. 关键步骤
[步骤1] → [步骤2] → [步骤3]...

## 4. 输出要求
[格式、风格、约束]
```

---

### 3. 专业优化 (user-prompt-professional)

**适用场景**: 需要精准描述的场景，添加量化标准和明确参数

**触发方式**:
```
专业优化：帮我处理数据
专业：写一份报告
```

**优化要点**:
- 泛泛而谈 → 具体要求
- 添加量化标准
- 明确参数和边界
- 可验收的标准

---

### 4-6. 上下文版优化

**触发方式**:
```
上下文优化：帮我写代码
上下文规划：做一个系统
上下文专业：处理这个业务
```

**特点**: 
- 在多轮对话上下文中使用
- 最小充分精炼
- 明确范围、参数、格式与验收标准

---

## 使用示例

### 示例1: 基础优化

**原始**:
> 帮我写个文章

**优化后**:
> 请写一篇关于AI技术发展的科普文章，要求：
> - 800-1000字
> - 面向普通读者
> - 包含3个实际应用案例
> - 语言生动有趣

---

### 示例2: 规划优化

**原始**:
> 帮我做一个旅行计划

**优化后**:
> # 任务：云南旅行计划
>
> ## 1. 角色与目标
> 你是一位专业旅行规划师，你的核心目标是设计一份7天云南深度游攻略。
>
> ## 2. 关键步骤
> 1. 昆明抵达与中转
> 2. 大理古城探索（2天）
> 3. 丽江古城与玉龙雪山（2天）
> 4. 泸沽湖环湖（2天）
> 5. 返回昆明
>
> ## 3. 输出要求
> - 每日行程安排
> - 推荐住宿与美食
> - 预算范围：3000-5000元

---

### 示例3: 专业优化

**原始**:
> 帮我处理数据

**优化后**:
> 请处理以下CSV数据文件：
> - 输入：data.csv（UTF-8编码）
> - 操作：统计每列非空值数量，计算数值列均值
> - 输出：result.json格式
> - 异常处理：跳过格式错误行并记录日志
> - 性能要求：处理10万行数据不超过10秒
